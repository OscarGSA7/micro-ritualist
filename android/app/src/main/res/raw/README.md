# Sonidos de Notificación

## Instrucciones

Coloca archivos de sonido .mp3 u .ogg en esta carpeta con los siguientes nombres:

| Archivo | Descripción |
|---------|-------------|
| `gentle.mp3` | Tono suave y calmado |
| `chime.mp3` | Campana relajante |
| `zen.mp3` | Sonido de cuenco tibetano |
| `nature.mp3` | Sonido de agua/naturaleza |
| `soft.mp3` | Tono minimalista y discreto |

## Dónde descargar sonidos gratis

### Opción 1: Pixabay (Recomendado)
- https://pixabay.com/sound-effects/search/notification/
- Licencia: Gratis para uso comercial y personal

### Opción 2: Freesound
- https://freesound.org/search/?q=gentle+notification
- Licencia: Varía (busca CC0 para libre uso)

### Opción 3: Mixkit
- https://mixkit.co/free-sound-effects/notification/
- Licencia: Gratis para cualquier uso

## Sonidos sugeridos de Pixabay:
1. **Gentle notification** - Busca "soft notification bell"
2. **Chime** - Busca "wind chime single"
3. **Zen** - Busca "singing bowl" o "tibetan bowl"
4. **Nature** - Busca "water drop notification" o "rain drop"
5. **Soft** - Busca "minimal notification" o "simple ding"

## Requisitos técnicos
- Formato: MP3 u OGG
- Duración: 1-3 segundos máximo
- Tamaño: < 100KB por archivo
- Nombres de archivo en minúsculas, sin espacios

## Después de agregar los sonidos
1. Ejecuta `flutter clean`
2. Ejecuta `flutter pub get`
3. Ejecuta `flutter run`
